========
Examples
========
All example code and meshes can be viewed by cloning the public repository:

.. code-block:: console

    $ git clone git://github.com/acreegan/pyvista_tools

then navigating to the examples folder:

.. code-block:: console

    $ cd pyvista_tools/examples
